Directives

    What are directives?

  • A directive is a JavaScript object that generate a view
    • You can reference to them by using HTML attribute
    • Built-in directives: ngRepeat, ngBind, ngModel, and ngClass

    Steps

    1. Create a directive
      • app.directive("dirName", functionality)
    2. Use directive
      • <dir-name></dir-name>
